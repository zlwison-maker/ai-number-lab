# AI Number Lab

AI Number Lab is a Next.js App Router site for generating entertainment-only phone number reports.

## Analytics

Google Analytics 4 is connected with the Google tag measurement ID `G-BYGKDVNZNK`.

- The tag is loaded globally from the root layout through `next/script`.
- Analytics is enabled only in production builds.
- Localhost hosts (`localhost`, `127.0.0.1`, `::1`, and `*.localhost`) do not load the tag or send data.
- App Router page changes are tracked by sending GA4 `page_view` updates when the pathname or query string changes.

## SEO

The production site URL is `https://numberlab.vip`.

- `/sitemap.xml` is generated by the App Router metadata route.
- `/robots.txt` allows all search engines and points to the production sitemap.
